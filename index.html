Q Game

C# .NET
Q Game is a user controlled puzzle game. Objective is to clear out all boxes in as few as moves as possible. Boxes are removed when the user selects a box and moves it to collide with a door of the same colour. Boxes move in a straight line and keep going in that direction until it collides with something. Players are able to design their own map in Q Game by selecting the design form in the Main Menu. After the user is satisfied with their map, they can save it and load it in the play form. has saved on their computer.
